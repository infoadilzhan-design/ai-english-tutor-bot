# AI English Tutor Bot 

This is an adaptive Telegram bot designed to help users learn English using AI

## Key Features:
 **Level Selection:** Tailored content for A1, A2, B1, and B2 levels.
 **Vocabulary Builder:** Generates word lists and examples in PDF format.
 **Grammar Guide:** Explains complex grammar topics with AI assistance
 **AI Podcasts:** Voice-generated podcasts for listening practice.
 **Interactive Quizzes:** Real-time Telegram polls to test your knowledge.

## Technologies Used:
 **Python** & **Aiogram 3** (Bot Framework)
 **OpenAI API** (Content Generation)
 **ReportLab** (PDF Generation)
 **gTTS** (Text-to-Speech)
